# Flight Operations & Delay Analysis Dashboard

## Project Overview

This Power BI project analyzes over 3 million airline flight records across 18 airlines and 8,000+ routes.

The dashboard provides insights into:

- Airline performance
- Flight operations
- Departure and arrival delays
- Flight cancellations
- Airport activity
- Route analysis

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## Dashboard Pages

### 1. Flight Operations Overview
- Total Flights
- Total Airlines
- Total Routes
- Monthly Flight Trends
- Top Airlines
- Airport Analysis

### 2. Delay & Cancellation Analysis
- Cancellation Rate
- Average Departure Delay
- Average Arrival Delay
- Top Delayed Airlines
- Delay Trends
- Cancelled Routes

## Key Insights

- Analyzed 3M+ flight records.
- Compared performance across 18 airlines.
- Evaluated delay patterns and cancellation trends.
- Identified high-traffic routes and airports.
